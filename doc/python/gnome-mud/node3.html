2 Basics

When GNOME-Mud launches, it scans the $HOME/.gnome-mud directory for Python scripts, ie. files ending with the suffix .py, and loads each script found in turn. Each script should import the GnomeMud module, which provides all the API functions necessary for writing useful scripts.
