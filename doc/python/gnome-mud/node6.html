4 Example GNOME-Mud script

Below is a simple example script that will replace occurrences of the text ``foo'' with ``bar'' in both MUD input and user commands, as well as printing the obligatory text to the display upon initialisation to identify its presence. 

import GnomeMud

def replace_foo(conn, data):
    return data.replace("foo","bar")

conn = GnomeMud.connection()
conn.write("Hello, world!\n")
GnomeMud.register_input_handler(replace_foo)
GnomeMud.register_output_handler(replace_foo)
